# Sudoku
Sudoku game


### Difficulty level
- easy
- medium / hard
- extreme
### Help level
- view only current digit
- view suggestions
- view tips
### Generate / solve features
- silent
- live
### Features
- next: generate or load new game
- clear: clear sudoku board
- confirm: save current layout
- back: return to the confirmed layout
- solve: solve current layout
- exit: leave the game
